# Hamza Javed — Portfolio

Personal portfolio website for **Hamza Javed**, AI Developer specializing in climate
analytics, smart agriculture, machine learning pipelines, and Python backends.

Built as a static site with a dark, glassmorphism theme. Hosted on GitHub Pages.

## Stack
- HTML, CSS (glassmorphism, gradients, scroll-reveal animations)
- Vanilla JavaScript
- Google Fonts (Sora, Inter)

## Local preview
Open `index.html` in any browser.

## Deployment (GitHub Pages)
1. Push to the `main` branch.
2. In the repo, go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to *Deploy from a branch*.
4. Select branch `main` and folder `/ (root)`, then **Save**.
5. The site will be live at `https://hamza1959.github.io/Portfolio/`.
